# 🚀 AI Background Remover

A modern and professional AI-powered background remover built using **HTML, CSS, and JavaScript** with integration of the **Remove.bg API**.

---

## ✨ Features

- 🖼️ Upload image (drag & drop supported)
- 🤖 AI-powered background removal
- ⚡ Fast processing with API
- 👀 Before & After preview
- 📥 Download transparent PNG
- 🎨 Premium Neon Red + Black UI
- 💻 Fully responsive design
- 🔄 Smooth animations and transitions

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Glassmorphism + Neon UI)
- JavaScript (Vanilla JS)
- Remove.bg API
